## Logic game
- Each level has 10 lanes. Cross 10 lanes to go to next level.
- Pass a level: *+10 pts*
- ***Collect a coin: +2 pts***
- Getting hit by a car: *die/restart*
- Getting hit by animals: *punishments/die*
